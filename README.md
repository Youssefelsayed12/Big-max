# ديجيتال ستور — نظام متكامل (متجر + لوحة تحكم + تسليم + دعم فني)

نظام حقيقي متكامل: سيرفر Node.js + قاعدة بيانات SQLite فعلية.
**بدون أي مكتبات خارجية** — مفيش أي `npm install` مطلوب، يشتغل فورًا.

## 🚀 طريقة التشغيل

```bash
node server.js
```

يحتاج **Node.js 22.5 أو أحدث** (لأن قاعدة البيانات SQLite مدمجة في Node نفسه).
تحقق من نسختك بـ: `node -v`

بعد التشغيل هتلاقي:
- 🛍️ المتجر: http://localhost:3000/
- 🔐 لوحة التحكم: http://localhost:3000/admin.html
- 📦 تتبع الطلب والدعم الفني: http://localhost:3000/track.html

## 🔑 بيانات الدخول الافتراضية للوحة التحكم

```
اسم المستخدم: admin
كلمة المرور:  admin123
```

⚠️ **غيّر كلمة المرور فورًا** من تبويب "الإعدادات" داخل لوحة التحكم بعد أول دخول.

## 📁 هيكل المشروع

```
digital-store/
├── server.js          # السيرفر الرئيسي (كل الـ API routes)
├── db.js              # قاعدة البيانات SQLite + البيانات الافتراضية
├── lib/
│   ├── auth.js         # تسجيل الدخول والجلسات (sessions)
│   └── util.js         # أدوات مساعدة (JSON, قراءة الطلبات, توليد الأكواد)
├── public/
│   ├── index.html       # واجهة المتجر (تجيب المنتجات من الـ API الحقيقي)
│   ├── admin.html        # لوحة التحكم الكاملة
│   └── track.html        # صفحة تتبع الطلب + الدعم الفني للعميل
└── store.db            # قاعدة البيانات (تتولد تلقائياً أول تشغيل)
```

## ✨ المميزات

**المتجر (index.html)**
- المنتجات بتتحمّل من قاعدة البيانات الحقيقية مباشرة
- عربة شراء + مودال دفع بيجمع بيانات العميل (اسم، إيميل، تليفون)
- عند تأكيد الطلب، بيتولد رقم طلب حقيقي (مثال: `ORD-35769AF91`) ويتخزن في قاعدة البيانات

**لوحة التحكم (admin.html)**
- تسجيل دخول محمي بجلسات حقيقية (sessions) في قاعدة البيانات
- نظرة عامة: إحصائيات مباشرة (عدد الطلبات، الإيرادات، التذاكر المفتوحة...)
- إدارة الطلبات: عرض كل الطلبات وتغيير حالتها (قيد المراجعة / جاري التجهيز / تم التسليم / ملغي)
- إدارة المنتجات: إضافة / تعديل / حذف / تفعيل - تعطيل منتجات
- الدعم الفني: عرض كل تذاكر العملاء والرد عليها مباشرة
- الإعدادات: اسم المتجر، العملة، بيانات التواصل، كود الخصم، وتغيير كلمة المرور

**تتبع الطلب (track.html) — صفحة العميل**
- العميل يدخل رقم الطلب + إيميله ويشوف حالة طلبه لحظيًا
- بعد ما الأدمن يحدد الطلب "تم التسليم" تظهر روابط التحميل
- تبويب منفصل لفتح تذكرة دعم فني أو متابعة رد سابق والرد عليه

## 🌐 لو عايز تنشر الموقع على الإنترنت (Deploy)

المشروع Node.js عادي، تقدر تنشره على أي استضافة تدعم Node.js زي:
- Render.com
- Railway.app
- VPS عادي (DigitalOcean, Hetzner...) مع PM2

فقط ارفع المجلد كامل وشغّل `node server.js` — مفيش خطوات تثبيت إضافية.

## 📝 ملاحظات مهمة

- قاعدة البيانات ملف واحد (`store.db`) بيتعمل تلقائيًا أول مرة تشغّل السيرفر
- لو عايز تبدأ من الصفر، امسح ملف `store.db` وشغّل السيرفر تاني
- روابط تحميل المنتجات بعد التسليم بتتحط يدويًا من لوحة التحكم حاليًا (تقدر تطورها لاحقًا لرفع ملفات تلقائي)
